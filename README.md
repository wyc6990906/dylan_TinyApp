## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- npm morgan
- Nodemon
- serve-favicon - custom favicon.ico

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `npm start` command.
